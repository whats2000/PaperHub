\begin{itemize}
\item Novel dual-memory architecture replaces inefficient iterative denoising.
\item Global and local memories stabilize execution and ensure temporal coherence.
\item Delivers 2.9x inference speedup with reduced computational steps.
\item Achieves a 98.6\% average success rate on the LIBERO benchmark.

\begin{frame}
\frametitle{Background \& Motivation}
\begin{itemize}
\item Standard Vision-Language-Action models map inputs to continuous actions via generative policies.
\item Iterative denoising from pure isotropic noise causes high latency and overhead.
\item Independent step-wise generation ignores history, leading to jittery execution.
\item Real-time control requires a mechanism to bypass pure noise and bridge temporal gaps.
\end{itemize}
\end{frame}
\begin{frame}
\frametitle{OptimusVLA: Framework Overview}
\begin{columns}
\begin{column}{0.48\textwidth}
\begin{itemize}
\item Vision-Language backbone extracts multimodal representations.
\item Global Prior Memory (GPM) retrieves semantic task priors.
\item Local Consistency Memory (LCM) models historical action sequences.
\item Memories initialize generation, successfully bypassing isotropic noise.
\end{itemize}

\begin{itemize}
\item LCM encodes recent executed actions for smooth temporal transitions.
\item Combines GPM task prior with LCM offset for final action generation.
\item Mitigates jittery movements common in standard VLA predictions.
\item Yields high execution stability with a reduced evaluation budget.
